# Minecraft Textures

## Required Textures
Add these Minecraft-style texture images (64x64 pixels, tileable):

1. **stone.png** - For card backgrounds
2. **grass.png** - For hero section
3. **wood.png** - For navbar
4. **bedrock.png** - For footer

## Where to Get Textures
- Extract from Minecraft resource packs
- Create your own pixelated textures
- Use free pixel art texture generators
- Download from texture sites (ensure licensing)

## Texture Specifications
- Size: 64x64 pixels (or multiples like 128x128)
- Format: PNG with transparency support
- Style: Pixelated, tileable patterns
- Optimize for web (compress file size)

## Usage
These textures will be used as background overlays with low opacity to give the Minecraft block feel to UI elements.
