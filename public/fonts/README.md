# Minecraft Fonts

## Required Font
Download "Minecrafter" or "Minecraft" font from:
- https://www.dafont.com/minecrafter.font
- https://www.fontspace.com/minecraft-font

Place the font file as `Minecrafter.ttf` in this directory.

## Alternative Free Fonts
If you can't find Minecrafter, you can use:
- Press Start 2P (Google Fonts)
- VT323 (Google Fonts)
- Silkscreen (Google Fonts)

These are pixelated fonts that give a similar retro gaming feel.
